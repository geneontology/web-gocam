GO-CAMs: linking GO Annotations

GO-Causal Activity Models (GO-CAMs) use a defined “grammar” for linking multiple GO annotations into larger models of biological function (such as “pathways”) in a semantically structured manner. GO-CAMs are created by expert biocurators from the GO Consortium, using the Noctua Curation Platform.

GO-CAMs: multiple uses

  • As “pathway diagrams” that can be browsed and visualized.
  • As full GO-CAM graphs that can be downloaded (as RDF Turtle) and downloaded at our GitHub repository.

  • As “reduced” pairwise GO annotations, included in the GO annotation files for backward compatibility with the broader GO communities.
